## 📌 Sobre o projeto

Este projeto foi desenvolvido com HTML e CSS com o objetivo de praticar a construção de layouts modernos e organização de conteúdo.

O tema aborda curiosidades sobre o sistema Android, apresentando informações de forma visual e estruturada.

## 🚀 Tecnologias utilizadas

* HTML (estrutura semântica da página)
* CSS (estilização, cores, layout e responsividade)

## 🎯 Funcionalidades

* Layout organizado e bem estruturado
* Uso de imagens e conteúdo informativo
* Estilização moderna com CSS
* Responsividade para diferentes dispositivos
